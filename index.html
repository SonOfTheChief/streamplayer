HOW TO USE IT

In fact, using this page is pretty simple. If you can't figure it out on your own, I have bad news for you.just kidding, you're a wonderful person

So here's a brief explanation:

  1. In the first box you can define which stream you want to use. You have two possibilities: Set your own or choose one of ours.
    2. Remember: The only possible sources are HLS and DASH.
  2. According to the type of stream you choose in step 1, only the players which can actually play the type of stream are going to be activated. Choose a player and enjoy.
